---
title: Getting Started
---

This guide helps you run the local Docusaurus dev server and add content.

1. Install dependencies (run in PowerShell):

```powershell
npm install
```

2. Start the development server:

```powershell
npm start
```

3. Add a new doc under `docs/` and update `sidebars.js` as needed.

4. To build the static site:

```powershell
npm run build
```
