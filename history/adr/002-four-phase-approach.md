# ADR 002 — Four-Phase Implementation Approach

Date: 2025-12-07

## Context
Project needs a clear, repeatable lifecycle for research-heavy textbook work: research → drafting → QA/review → finalize/deploy.

## Decision
Adopt the Four-Phase approach: Phase 0 (Research & Source Collection), Phase 1 (Drafting), Phase 2 (QA & Review), Phase 3 (Finalization & Deployment).

## Consequences
- Produces measurable checkpoints and deliverables.
- Simplifies scheduling and resource allocation.

## Status
Accepted
